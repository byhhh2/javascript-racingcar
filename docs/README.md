## 🚀 구현할 기능 목록

### 경주 

- [ ] 하나의 턴(자동차의 전진 혹은 정지)마다 1초의 텀을 둔다. 
- [ ] 1초의 텀 동안 로딩 애니메이션을 자동차의 위치를 표시하는 화살표가 보여질 부분에 보여준다.

### 결과 

- [ ] 게임이 끝난 후 결과를 보여준 후, 2초 뒤 축하하는 alert를 띄운다.
